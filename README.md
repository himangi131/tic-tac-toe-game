# # tic-tac-toe-game
This is a beginner-friendly Tic Tac Toe game built using HTML, CSS, and JavaScript as part of my learning journey in JavaScript.

The goal of this project was to understand core JavaScript concepts like DOM manipulation, event handling, and game logic implementation in a simple and interactive way.

#✨ Features
🎯 Two-player gameplay (X vs O)
🎨 Colorful and modern user interface
🎈 Balloon animation when a player wins
🔊 Sound effect on winning
🔁 Restart game functionality
⚡ Smooth interactions and responsive design
#🧠 What I Learned
How to manipulate the DOM using JavaScript
Handling user events like clicks
Managing game state using arrays and variables
Writing logic to check winning conditions
Improving UI/UX with CSS animations
#🛠️ Technologies Used
HTML
CSS
JavaScript
🚀 How to Run the Project
Download or clone this repository
Open the index.html file in your browser
Start playing the game 🎉
